# TrMIS: Tuberculosis Training Management Information System

**Role:** Technical Program Manager | AI & Cloud Integration

TrMIS is a national-level platform designed to **streamline tuberculosis training workflows**, provide actionable insights, and improve training efficiency in healthcare. Developed in collaboration with InNeed Intelligent Cloud and JSI, it leverages cloud infrastructure and data-driven solutions.

## Key Features
- Centralized training management and tracking
- AI-driven analytics for training outcomes
- Workflow automation for trainers and administrators
- Scalable cloud deployment using **AWS/GCP** and containerization (**Docker/Kubernetes**)
